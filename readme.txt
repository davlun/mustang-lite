Mustang Lite theme lets you create responsive business websites.


== Description ==

Mustang Lite theme lets you create responsive business websites. It is suitable for portfolio, business websites, personal presentations and much more. The theme is customizable so you can make it your own. As the theme is translation ready and supports right-to-left (RTL) languages, you can successfully built a multilingual website with it.
